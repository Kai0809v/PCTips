## 所在位置
系统的终端使用的一般是profile.ps1配置脚本，如果没有就创建一个

```Bash
C:\Users\Kai\Documents\WindowsPowerShell
```
```Bash
# 记事本打开
notepad $PROFILE
# VS code打开，但是打开的是 `Microsoft.PowerShell_profile.ps1` 文件，不加载这个
code $PROFILE
```
记事本通常是UTF-8，但是终端脚本默认使用GBK编码，不管你使用下面设置，在脚本里面的中文还是乱码，这些只在启动后生效
```Bash
# Set console encoding to UTF-8
[Console]::OutputEncoding = [System.Text.Encoding]::UTF8
[Console]::InputEncoding = [System.Text.Encoding]::UTF8
$OutputEncoding = [System.Text.Encoding]::UTF8
```
## 终端输出Logo
简单的办法就是 `Write-Host` ，找一个 `文本转ASCII艺术字Text to ASCII Art Generator (TAAG)` 就行  
别忘了可以选格式，比如C/C++注释类型，这样好替换

## 我的配置
- 我的win11家庭版要用GB类编码保存
```Bash

# 设置输出编码为UTF-8

# 欢迎信息
Write-Host "===================================" -ForegroundColor DarkGray

Write-Host " Now:   $(Get-Date)" -ForegroundColor Yellow

Write-Host " 要使用conda的话，用下面这个虚拟环境： " -ForegroundColor Blue
```

## 关于`-ForegroundColor`的色彩表
| 分类 | 英文名称 | 中文名称 | 适用场景 |
|------|----------|----------|----------|
| 基础色 | Black | 黑色 | 背景首选 |
| 基础色 | White | 白色 | 文字首选 |
| 基础色 | Red | 红色 | 警告/错误提示 |
| 基础色 | Green | 绿色 | 成功/提示符 |
| 基础色 | Blue | 蓝色 | 普通提示 |
| 基础色 | Cyan | 青色 | 标题/分隔线 |
| 基础色 | Yellow | 黄色 | 路径/高亮 |
| 基础色 | Magenta | 品红色 | 强调文字 |
| 深色系 | DarkRed | 深红色 | 低调警告 |
| 深色系 | DarkGreen | 深绿色 | 低调成功 |
| 深色系 | DarkBlue | 深蓝色 | 低调文字 |
| 深色系 | DarkCyan | 深青色 | 低调标题 |
| 深色系 | DarkYellow | 深黄色 | 低调高亮 |
| 深色系 | DarkMagenta | 深品红 | 低调强调 |
| 深色系 | DarkGray | 深灰色 | 次要文字 |
| 深色系 | Gray | 浅灰色 | 注释文字 |
